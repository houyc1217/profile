Built with a minimalist Apple-style theme, featuring:
- 🌐 **Web-only** design
- 🎨 Low-saturation blue palette
- ⚙️ Easy navigation: About | Education | Experience | Projects | Contact
- 📱 Fully responsive design
- ⬇️ Animated scroll indicator
